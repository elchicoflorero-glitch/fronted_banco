# Módulos

Los módulos contienen componentes complejos y reutilizables específicos de características del negocio.

## Estructura

Cada módulo debe tener su propia carpeta con:
- Componentes principales
- Hooks específicos del módulo
- Tipos/interfaces
- Servicios específicos

## Ejemplos

- `clients/`: Módulo de gestión de clientes
- `accounts/`: Módulo de gestión de cuentas
- `transactions/`: Módulo de transacciones
